# CyberSafe Home

**Affordable cybersecurity audits and remediation for small businesses and individuals**

![CyberSafe Home Logo](assets/logo/cybersafe-home-logo.png)

CyberSafe Home helps non-technical owners protect their customer data, devices, and online presence without expensive IT consultants. We start with a free 15-question audit, deliver personalized reports, and offer paid packages with consultations and monitoring.

### Live Site
https://cybersafehome.org

### Current Status (February 2026)
- MVP launched  
- Free audit live (Google Forms + auto-scoring)  
- First paying customers acquired via local outreach  
- Actively iterating on conversion and report delivery
